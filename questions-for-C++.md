圆的面积和周长

题目描述
给定一个圆的半径，输出圆的面积和周长。圆周率取值PI=3.1415926。

输入描述
输入一个整数r(0 < r < 1000)

输出描述
输出两行，第一行是圆的面积，第二行是圆的周长，保留三位有效数字

样例输入
2

样例输出
131
31313

2. 

水仙花数

题目描述
给定一个数字，如果每个分位数字的立方还等于该数字，那么就是这个数就是水仙花数。
例如，数字153， 1^3 + 5^3 +3^3 = 1 + 125 + 27 = 153, 因此这个数就是水仙花数。
数字12, 1^3+2^3 != 13, 12不是水仙花数。

请输出所有小于10000的水仙花数。

输入描述
无输入

输出描述
输出所有小于10000的水仙花数，每个数字一行。

样例输入
无

样例输出
无
